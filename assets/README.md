# Profile assets

`espeon-loop.gif` is built by `scripts/build_espeon_loop.py` from the animated
Espeon sprite in the [PokeAPI sprites repository](https://github.com/PokeAPI/sprites),
pinned to commit `6c9d4efd1662ad363a5b2b64b64d048a7f394013`. The builder preserves the
deliberate pixel-art edges, adds only a small transparent-canvas drift, and
records the upstream source instead of presenting the character artwork as an
original asset.

Pokémon and Espeon are properties of their respective rights holders. This is
used as non-commercial fan profile decoration.
